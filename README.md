# Clínica Eras - Estética & Saúde

Site institucional da **Clínica Eras**, localizada em Alagoinhas-BA, desenvolvido com HTML, CSS e JavaScript puro.

## Sobre o Projeto

Landing page responsiva para a Clínica Eras, oferecendo informações sobre serviços de saúde e estética, estrutura da clínica e canais de contato/agendamento.

## Funcionalidades

- Navegação responsiva com menu hamburguer para mobile
- Seções: Início, Sobre, Serviços, Consultas, Estrutura, Comunidade e Contato
- Design elegante com as fontes Playfair Display e Inter
- Ícones via Font Awesome

## Serviços Apresentados

- Pilates
- Hidroginástica
- Hidroterapia
- Fisioterapia
- RPG (Reeducação Postural Global)

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- [Font Awesome 6.5](https://fontawesome.com/)
- [Google Fonts](https://fonts.google.com/) (Playfair Display & Inter)

Basta abrir o arquivo `index.html` diretamente no navegador — não requer servidor ou instalação de dependências.
